## Mastodon for iOS Keyboard Shortcuts
| Action                         | Key        |
|:------------------------------:|:----------:|
| Home Timeline/Refresh Timeline | ⌘ + 1      |
| Open Compose Window            | ⌘ + N      |
| Send Toot                      | ⌘ + Return |
| Explore Tab                    | ⌘ + 2      |
| Notifications                  | ⌘ + 3      |
| Settings                       | ⌘ + ,      |
| Favorites                      | ⌘ + F      |
